Project that I made for job interview

Simple inventory system and scalable item system

______________________________________

Unity Version: 2022.3.5f1

Grid textures are from Asset Store, everythink else is made by me


Quick instructions:

WASD -> movement
ctrl -> crouch
ctrl + W -> slide
space -> jump

Tab or i -> open inventory
1,2,3 etc or scroll -> choose item
LMB -> use item that is currently in your hand
